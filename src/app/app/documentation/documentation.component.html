Introduction

Setup and install Angular and GMAD

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install angular-cli

NPM

$ npm install @angular-cli

Yarn

$ yarn install angular-cli

Install GMAD

$ npm install @gmad

Create a new angular app

$ ng new angularfire

Open a development server

$ ng serve //optional -o

Optional: Generate an Componenent

$ ng g c Home

Optional: Generate an Service

$ ng g service Home
